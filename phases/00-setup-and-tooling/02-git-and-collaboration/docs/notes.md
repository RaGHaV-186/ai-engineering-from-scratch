# Phase 00, Lesson 02 — Git & Collaboration

## The four-stage model
1. Working directory — files you're editing
2. Staging area — draft of your next snapshot (git add)
3. Local repo — permanent history on your machine (.git folder)
4. GitHub — backup, portfolio, collaboration hub

## Key commands
| Command | What it does |
|---|---|
| `git status` | Show what changed |
| `git add <file>` | Stage a specific file |
| `git add .` | Stage everything |
| `git diff --staged` | Review what you're about to commit |
| `git commit -m "msg"` | Create a snapshot |
| `git push` | Send to GitHub |
| `git log --oneline` | See commit history |
| `git branch <name>` | Create a branch |
| `git checkout -b <name>` | Create + switch to branch |
| `git merge <name>` | Merge branch into current |
